# Fifa-Dashboard
1. Project Title
GoalScope Analytics: FIFA Tournament Performance Dashboard
An interactive sports analytics dashboard built in Power BI to analyze FIFA tournament matches, team performance, scoring patterns, and historical trends across competitions.

2. Short Description
The FIFA Analytics Dashboard is an interactive Power BI report designed to explore historical FIFA tournament data and uncover insights about team performance, goal statistics, match outcomes, and tournament trends.
This dashboard helps sports analysts, fans, and strategists quickly understand patterns in international football competitions.

Source: Public FIFA match datasets (historical international football match records)
Dataset contains:
Match results
Goals scored
Home & away teams
Tournament year
Stadium & host country
Winners and match statistics
The data represents multiple FIFA tournaments across different years enabling trend analysis and team comparison.
